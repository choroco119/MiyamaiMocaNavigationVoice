# Euro Truck Simulator 2 (ETS2) - 宮舞モカ ナビ音声MOD 🚚🧭

音声合成ソフト **VOICEPEAK 宮舞モカ**（および愛猫のワトソンくん）の音声を使用した、Euro Truck Simulator 2 (ETS2) 用のカスタムGPSボイスナビゲーションMODです。

ちょっと陰キャで自信なさげ、でも健気で可愛い宮舞モカ（高校2年生）が、あなたの長距離トラック輸送の旅をフルボイスで優しくサポートします。愛猫のワトソンくんに話しかけながらルート案内する、モカならではのセリフが盛りだくさんです！

---

## ✨ 特徴
*   **高品質な音声アセット:** VOICEPEAK 宮舞モカのクリアな音声データを使用。
*   **最新のFMOD仕様に完全準拠:** ETS2のバージョン1.37以降で導入された FMOD サウンドシステムに完全対応しています。
*   **すべての音声イベントを網羅:** 
    *   基本の案内（右左折、Uターン、案内開始・終了）
    *   分岐や維持（高速出口、右左維持）
    *   ラウンドアバウト（1〜6番目のすべての出口）
    *   速度制限注意および速度オーバー時の警告（タジタジになりながら叫びます）
    *   2段階の複合指示（「その先、左折です」など）もタイムライン合成によりエラーなしで完全再生。

## 🚀 導入手順（インストール方法）

1.  **MODファイルのダウンロード:**
    *   リポジトリ右側の [Releases](https://github.com/choroco119/ets2_voice_mod_miyamai_moca/releases) ページから、最新バージョンの **`Miyamai Moca Navigation Voice.scs`** をダウンロードします。
2.  **MODフォルダへの配置:**
    *   ダウンロードしたファイルを、お使いのPCの以下のフォルダに配置します：
        `C:\Users\<ユーザー名>\Documents\Euro Truck Simulator 2\mod\`
        (※OneDriveが有効な環境の場合は、`OneDrive\ドキュメント\Euro Truck Simulator 2\mod\` に配置してください)
3.  **MODマネージャーでの有効化:**
    *   ETS2を起動し、プロファイル選択画面の「MODマネージャー」を開きます。
    *   一覧から **`Miyamai Moca Navigation Voice`** を選択し、「有効（アクティブ）」に設定して変更を適用します。
4.  **ゲーム内オプションでの選択:**
    *   ゲーム内メニューの **「オプション」 -> 「オーディオ」** を開きます。
    *   「ボイスナビゲーション」の設定項目で、言語選択リストから **`日本語 - 宮舞モカ`** を選択します。
    *   「プレビュー」ボタンを押し、モカの声（「ルートの案内を開始します……っ！」）が再生されれば導入成功です！

## 🛠️ 開発者向け情報 (ビルド方法)
このMODは、FMOD Studio を使用してビルドされたサウンドバンクを含んでいます。

*   **FMOD Studioプロジェクト:** `assets/template/` ディレクトリ内に格納されています。
*   **自動パッケージング:**
    *   FMOD Studioでバンクをビルドし、出力された `mym_voice.bank` と `mym_voice.bank.guids` を `mod_root/sound/navigation/` に上書き配置した後、ルートにある `make_mod.bat` を実行すると、自動で無圧縮の `.scs` パッケージが再構築されます。
