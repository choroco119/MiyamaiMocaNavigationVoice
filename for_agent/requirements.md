# Euro Truck Simulator 2 (ETS2) ナビ音声MOD 作成要件定義書

この要件定義書は、音声合成ソフト「VOICEPEAK 宮舞モカ」を使用し、ETS2用のカスタムボイスナビゲーションMODを3パターンのランダム再生バリエーションで作成するための仕様を定義します。

---

## 1. 概要
- **対象ゲーム:** Euro Truck Simulator 2 (ETS2) バージョン1.37以降 (FMOD サウンドシステム対応)
- **音声キャラクター:** VOICEPEAK 宮舞モカ (および愛猫のワトソンくん)
- **仕様特徴:** 各案内イベントに **3つの異なるセリフパターン** を用意し、FMOD Studioの「Multi Sound」機能を使用してゲーム内でランダムに再生させます。セリフから言い淀みや末尾の吃音（「……っ」など）を完全に排除し、滑らかで自然な発音を実現します。

---

## 2. 音声アセットの基本仕様
- **ファイルフォーマット:** WAV形式
- **サンプリングレート:** 44,100 Hz
- **ビット深度:** 16-bit
- **チャンネル数:** モノラル (Mono)
- **ファイル命名規則:** `[イベント名]_[パターン番号].wav` (例: `turn_left_1.wav`, `turn_left_2.wav`, `turn_left_3.wav`)

## 4. 実装フロー
1. **音声素材の準備:** 上記イベントに対応する音声データ（WAV推奨、44.1kHz / 16bit / Mono）を `assets/` に準備します。
2. **FMOD Studioの準備:** SCS公式 of FMODテンプレート（またはボイスナビゲーション用テンプレート）を用意し、FMOD Studio（バージョンはSCS指定のもの）を導入します。
3. **イベント割り当て:** FMOD Studioで各音声イベントに対して、準備した音声を紐付けます。ランダム化のため、各イベントを「Multi Sound」に設定します。
4. **バンクの構築:** FMOD Studioから `.bank` と `.bank.guids` を出力します。
5. **定義ファイルの作成:** `.sii` ファイル（ナビゲーション定義）と `manifest.sii` を作成します（任意で、MODマネージャー用アイコン画像 `276x162` JPG を配置して `manifest.sii` で指定することも可能です）。
6. **パッケージング:** `mod_root/` 以下のファイルを `make_mod.bat` でパッケージングし、無圧縮の `.scs` アーカイブ（中身は無圧縮ZIP）として、ETS2の `mod` フォルダに配置します。
