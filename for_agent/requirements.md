# ETS2 ナビ音声MOD作成 要件定義書

本ドキュメントは、Euro Truck Simulator 2 (ETS2) 向けカスタムナビ音声MODを作成するための要件定義および技術仕様書です。

## 1. 目的
ETS2のゲーム内で利用可能な、オリジナルのナビゲーション音声MOD（Voice Navigation MOD）を作成します。
ユーザーが指定する音声アセット（または音声合成等によって生成された音声アセット）をゲームに組み込めるようにします。

## 2. 対象環境・動作要件
- **対象ゲーム:** Euro Truck Simulator 2 (バージョン 1.37 以降)
- **サウンドエンジン:** FMOD Sound System (FMOD Studio 2.xx 系統を使用)
- **プラットフォーム:** Windows

## 3. MOD of 全体構成
ETS2の1.37以降では、音声は FMOD Studio でビルドされた `.bank` ファイルとして提供する必要があります。

```
ets2_voice_mod/
├── for_agent/
│   └── requirements.md        # 本要件定義書
├── Plan.md                    # 実装計画書
├── AICHANGELOG.md             # 変更履歴書
├── assets/                    # 音声素材（WAV/OGG等）を格納するディレクトリ
├── fmod_project/              # FMOD Studioのプロジェクトテンプレートを配置するディレクトリ
└── mod_root/                  # MODパッケージング用フォルダ
    ├── manifest.sii           # MODのマニフェストファイル
    ├── material/              # （任意）MODのアイコン画像など
    └── sound/
        └── navigation/
            ├── <voice_name>.sii      # ナビ音声設定ファイル
            ├── <voice_name>.bank     # FMODでビルドされたバンクファイル
            └── <voice_name>.bank.guids # FMODで出力されたGUIDリストファイル
```

## 5. 実装フロー
1. **音声素材の準備:** 上記イベントに対応する音声データ（WAV推奨、44.1kHz / 16bit / Mono）を `assets/` に準備します。
2. **FMOD Studioの準備:** SCS公式のFMODテンプレート（またはボイスナビゲーション用テンプレート）を用意し、FMOD Studio（バージョンはSCS指定のもの）を導入します。
3. **イベント割り当て:** FMOD Studioで各音声イベントに対して、準備した音声を紐付けます。
4. **バンクの構築:** FMOD Studioから `.bank` と `.bank.guids` を出力します。
5. **定義ファイルの作成:** `.sii` ファイル（ナビゲーション定義）と `manifest.sii` を作成します（任意で、MODマネージャー用アイコン画像 `276x162` JPG を配置して `manifest.sii` で指定することも可能です）。
6. **パッケージング:** `mod_root/` 以下のファイルを `make_mod.bat` でパッケージングし、無圧縮の `.scs` アーカイブ（中身は無圧縮ZIP）として、ETS2の `mod` フォルダに配置します。
